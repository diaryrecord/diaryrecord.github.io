# 2019-11-03 Sunday 一个颓废的周末



***By Long Luo***

今天是2019年11月03日，星期天。

--------------

本来想晚上早点睡，早上8点半起床去健身房锻炼的，结果晚上睡不着，只好起来看书上网。

很不幸又玩了几局斗地主，在b站看了几期视频，终于有点睡意开始睡觉。

早上一直睡到10点，起来后叫外卖。

睡到2点去车公庙打球，但是白跑一趟，时间浪费了一个半小时。

回来后本来想去游泳，然后去深圳湾徒步的。

但是回来后下载电子书，然后开始玩斗地主，从4点一直晃悠到晚上7点。

好说歹说终于去游泳，游了1个多小时到9点多。

回来后又看电视，然后又玩斗地主到2点才睡，真是非常颓废的一天！
