# 2020-01-05-Feb-Go-To-Hospital


今天是2021年01月05日，5/365。

早上起来后，决定去北大深圳医院看病。

8点20出门，地铁上好拥挤，在9点20到达医院。

医院里人实在是太多了。

在国内一定要好好保养好身体，因为医院里人实在是太多了！

本来是9:30的号，结果等到10点多才看到病。

我向医生描述了下我的症状，然后我说我怀疑有前列腺炎，医生然后说公司体检B超测不出来。

于是医生给我开了几个检查。

由于上午B超的号已经没了，所以我就跑到外科楼影像中心去检查，但是那边上午也没有号了，只有下午才有号。

11点我跑去附近的萨莉亚吃了个饭，然后买了瓶水，回医院休息。

2点开始先检查了一次，做B超的是个可爱的妹子，但是可能水喝的不多，于是我又去买了瓶水，这次快憋不住了再去检查。

第一次检查时，医生说怀疑精囊腺炎，当时听了有点害怕，我还没结婚生子呢！

然后排尿完了之后又测了一次，这次测的是残尿，发现还是有点多的。

B超做完了之后，我看了下报告，当时有点紧张，上面我当时看了觉得挺严重的。

拿了报告给医生看了之后，医生说是慢性前列腺炎，给了开了药，一个月后复诊。

拿药的时候才发现，医生给我开了一个中成药，我赶紧问怎么退。

这个医生有点坑，故意开中成药给我。

退完药之后就回去了。

到家了之后带上电脑去附件的星巴克学习到9点就回去了。

晚上在楼下走了6圈，现在要多锻炼身体。

先安顿好自己的身体，再求变！

2021目前还没有找到工作！
