# 2020-01-02-Sat-A-Breaking-Day


今天是2021年01月02日，2/365。

因为昨天晚上约了xm来家吃午饭，早上8点多起床了。

起床了之后先下去转了一圈，然后上了会网，做了会编程题。

xm直到12点多才到，她今天穿了一条象牙色裤子，一件v领毛线衣，披着一件蓝色毛呢外套。

接ta到家之后，因为我忘了买饮料，然后我下去买了瓶椰汁和橙汁，本来应该早就买的。

不过吃饭的时候我们没吃多少。

吃完午饭已经2点多了。

我们然后一起看电视。

下午我吻了她。

5点多的时候，我们出发前往西湾红树林，看看落日。

一路聊了很多。

7点多回来了，然后决定一起做蛋炒饭吃。

一起去买了葱、火腿肠、香菜，一起做了一个蛋炒饭。

不过煎鸡蛋时，油放的太多了，导致蛋炒饭太油腻了，只吃了一小碗就吃不下了。

然后一起吃了点蓝莓。

然后就10点啦，我送她到了五和地铁站，由于时间太晚，本来应该送她到家的，但是时间来不及了。

就这样，就结束了2021年的第二天。
