# 2020-05-03 Sunday



***By Long Luo***

今天是2020.05.03，周日。

--------------

今天是2020.05.03，假期第三天。

上午8点起床，然后去了龙华上塘地铁站看房。

因为昨晚约了以前认识的一位妹子去看房。

金茂府豪宅，去了营销中心，但看房需要验资300万。

看了下户型图，然后出来了。

之后一起打的去了观澜的龙光玖誉府，结果发现已经全部卖完了。

中途手机还丢在滴滴车上，然后重新找回来了。

最后又重新回到上塘荟吃饭，商场里人很少，一片萧条景象。

很多餐饮业服务员比顾客还多。

这个妹子从肢体需要来看，对我挺抵触的，还是放弃吧！

吃完潮汕牛肉火锅之后，说声再见就回家了。

回家以后午休了一个小时。

然后出发去看门，想换个门。

转了几个地方，发现有些店主太坑人了。

如果你不知道网上价格，你就被活活坑死，做生意真的就靠信息不对称赚钱啊！
